A hands-on Active Directory home lab built using Windows Server 2019 and Windows 11 to practise real-world IT Support and Service Desk tasks. The lab focuses on user account management, Group Policy configuration, and troubleshooting common domain-related issues in a controlled internal network.

### Key skills shown
Active Directory user and group management
Group Policy configuration and troubleshooting (user vs computer scope)
Domain join and authentication troubleshooting
DNS awareness and trust relationship recovery
Structured IT Support problem-solving and documentation

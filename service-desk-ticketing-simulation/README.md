A Service Desk ticketing simulation built using Jira Service Management and a small Windows home lab. The project focuses on day-to-day IT Support tasks such as logging and prioritising tickets, troubleshooting common issues, communicating with users, and closing incidents correctly.

### Key skills shown
Jira Service Management (ITSM) ticket handling
Incident prioritisation and categorisation
Active Directory access and login troubleshooting
Windows desktop and printer support
Network connectivity diagnostics
Professional user communication and documentation

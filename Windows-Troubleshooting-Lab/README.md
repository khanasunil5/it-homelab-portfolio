A hands-on Windows troubleshooting lab built in a domain environment to practise common Service Desk and IT Support issues. The lab focuses on diagnosing and resolving realistic end-user and system problems on a Windows 11 client joined to an Active Directory domain.

### Key skills shown
Windows desktop and user profile troubleshooting
DNS and network connectivity diagnosis in a domain environment
Windows services troubleshooting (Search, Print Spooler, Windows Update)
NTFS and shared folder permission analysis
Event Viewer log review and root cause identification
Structured Service Desk-style investigation and documentation

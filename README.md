## Labs Included

### Active Directory Home Lab (IT Support / Service Desk)
A hands-on Active Directory home lab built using Windows Server 2019 and Windows 11 to practise real-world IT Support and Service Desk tasks. The lab focuses on user account management, Group Policy configuration, and troubleshooting common domain-related issues in a controlled internal network.

### Key skills shown
Active Directory user and group management
Group Policy configuration and troubleshooting (user vs computer scope)
Domain join and authentication troubleshooting
DNS awareness and trust relationship recovery
Structured IT Support problem-solving and documentation

### Full lab repository
[active directory lab](https://github.com/khanasunil5/active-directory-home-lab)

### Service Desk Ticketing Simulation (Jira ITSM)
A Service Desk ticketing simulation built using Jira Service Management and a small Windows home lab. The project focuses on day-to-day IT Support tasks such as logging and prioritising tickets, troubleshooting common issues, communicating with users, and closing incidents correctly.

### Key skills shown
Jira Service Management (ITSM) ticket handling
Incident prioritisation and categorisation
Active Directory access and login troubleshooting
Windows desktop and printer support
Network connectivity diagnostics
Professional user communication and documentation

### Full project repository
[service-desk-ticketing-simulation](https://github.com/khanasunil5/service-desk-ticketing-simulation)

---

### Windows Troubleshooting Lab (Domain Environment)
A hands-on Windows troubleshooting lab built in a domain environment to practise common Service Desk and IT Support issues. The lab focuses on diagnosing and resolving realistic end-user and system problems on a Windows 11 client joined to an Active Directory domain.

### Key skills shown
- Windows desktop and user profile troubleshooting
- DNS and network connectivity diagnosis in a domain environment
- Windows services troubleshooting (Search, Print Spooler, Windows Update)
- NTFS and shared folder permission analysis
- Event Viewer log review and root cause identification
- Structured Service Desk-style investigation and documentation

### Full lab repository
[Windows Troubleshooting Lab](https://github.com/khanasunil5/Windows-Troubleshooting-Lab)
